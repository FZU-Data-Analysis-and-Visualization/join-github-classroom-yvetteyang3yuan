### 数学公式

公式说明：均值和方差是数据分析中常用的描述性统计指标，分别用于反映数据的平均水平和离散程度。

均值公式：

$$
\bar{x} = \frac{1}{n} \sum_{i=1}^{n} x_i
$$

方差公式：

$$
\sigma^2 = \frac{1}{n} \sum_{i=1}^{n} (x_i - \bar{x})^2
$$

## 4.Python分析代码

为了找出 2025 赛季 GSV 球员中场均得分排名前 8 的球员，本文使用 Python 读取 Excel 数据，并按照场均得分 PTS 进行降序排序。

```python
import pandas as pd
# 读取Excel数据
df = pd.read_excel("12.2025赛季GSV场上总数据.xlsx", 
                   sheet_name="具体数据 做雷达图分析")

# 按场均得分PTS降序排序，选出前8名球员姓名
top8_names = df.sort_values("PTS", ascending=False).head(8)["Name"]

print("场均得分Top8球员：")
print(top8_names.to_string(index=False))
```

由代码运行结果可知，Kayla Thornton 是 2025 赛季 GSV 队内场均得分最高的球员，Veronica Burton、Tiffany Hayes 等球员也位居得分榜前列，说明她们是球队主要的进攻贡献者。
